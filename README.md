 Sistema de Biblioteca

 Descrição
Este projeto consiste no desenvolvimento de um sistema de biblioteca em Python, com o objetivo de gerenciar informações relacionadas a pessoas, usuários, funcionários, livros e empréstimos.

Objetivo
Aplicar conceitos de programação, como organização de código, lógica e uso de ferramentas de desenvolvimento, por meio da criação de um sistema funcional.

Funcionalidades
O sistema possui as seguintes funcionalidades:

- Cadastro de pessoas  
- Cadastro de usuários  
- Cadastro de funcionários  
- Cadastro de livros  
- Empréstimo de livros  

Tecnologias utilizadas
- Python  
- GitHub  
- Visual Studio Code

Extensões utilizadas no Visual Studio Code:
-  Python
- Python Debugger
- Pylance
- Python environments

A pasta `data/` é utilizada para armazenar os dados do sistema, simulando um banco de dados.
